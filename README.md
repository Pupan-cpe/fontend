## Instalation 

1. Clone repository
```shell
git clone https://github.com/flatlogic/material-vue.git
```
2. Get in the project folder
```shell
cd Pupan-cloud
```
3. Install dependencies via npm or yarn
```shell
npm install
```

## Quick start
Run development server
```shell
npm run serve
```
